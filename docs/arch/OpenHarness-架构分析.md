# OpenHarness 架构分析

## 1. 职责概述

OpenHarness 是一个轻量级的AI代理基础设施，提供工具使用、技能管理、内存管理和多代理协调等核心功能。它的主要职责包括：

- **代理循环管理**：处理流式工具调用循环、API 重试、并行工具执行和令牌计数
- **工具集管理**：提供 43+ 种工具，包括文件 I/O、shell、搜索、Web 和 MCP 等
- **技能系统**：支持按需加载技能（.md 文件），兼容 anthropics/skills
- **插件生态**：兼容 claude-code 插件，支持命令、钩子、代理和 MCP 服务器
- **权限管理**：实现多级权限模式、路径级规则和命令拒绝列表
- **多代理协调**：支持子代理生成和委托、团队注册和任务管理
- **个人代理应用**：提供 ohmo 个人 AI 代理，支持 Feishu/Slack/Telegram/Discord 等平台
